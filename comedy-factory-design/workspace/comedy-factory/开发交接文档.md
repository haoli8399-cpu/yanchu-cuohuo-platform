## 1. 项目概述

### 1.1 项目名称

**喜剧工厂 (Comedy Factory)** -- B2B 演出内容交易平台微信小程序

### 1.2 业务模式

连接"活动公司"（需方）与"演员"（供方）的 B2B 演出内容撮合平台。活动公司可以浏览演出方案、提交需求、查看报价、确认签约；演员可以查看排期、确认接单、签到打卡、查看结算。

### 1.3 用户角色

| 角色 | 说明 | 入口页面 |
|------|------|----------|
| 活动公司 | 发布演出需求、浏览方案、管理订单 | 发现首页 (discover) |
| 演员 | 查看排期、确认演出、签到、结算 | 排期列表 (assignment-list) |
| 未注册用户 | 微信登录后根据角色分配 | 登录页 (login) |

### 1.4 技术栈

| 类别 | 选型 |
|------|------|
| 运行平台 | 微信小程序 |
| UI 框架 | Vant Weapp（推荐） |
| 跨端框架 | uni-app（推荐） |
| 设计稿宽度 | 375px |
| 页面结构 | 16 个 HTML 原型页面 |

## 4. 页面详细说明

### 4.1 登录页 -- `pages/login/index`

**页面用途**: 用户首次进入小程序的登录页面，支持微信一键登录。

**布局结构** (从上到下):
1. **紫色渐变顶部区域** (45vh 高度): `linear-gradient(170deg, #7c3aed 0%, #5b21b6 100%)`，内含背景图 (半透明叠加) + 品牌名称"喜剧工厂" (32px bold 白色) + 标语"连接优质演出内容" (14px 70%白色)
2. **白色底部卡片**: 圆角 `24px 24px 0 0`，`margin-top: -32px`，包含:
   - 微信一键登录按钮: 绿色 `#07C160`，全圆角胶囊，高度 48px，带阴影
   - 协议行: "用户协议" / "隐私政策" 链接

**关键组件**: 微信登录按钮、协议文字链接

**交互说明**:
- 点击"微信一键登录" -> 调用 `wx.login()` -> 登录成功跳转发现首页
- 协议链接 -> 跳转 H5 协议页
- 底部安全区: `padding-bottom: env(safe-area-inset-bottom)`

---

### 4.2 发现首页 -- `pages/discover/index`

**页面用途**: 活动公司端的首页，展示 Banner、快捷入口、热门推荐。

**布局结构** (从上到下):
1. **状态栏安全区**: `env(safe-area-inset-top)`
2. **Banner 轮播**: 16:9 比例，圆角 16px，底部渐变遮罩，分页指示点（激活态 4x1.5 圆角紫色）
3. **快捷入口网格** (2x2): 4 个入口卡片
   - 选方案 (紫色图标 + `#f5f3ff` 背景)
   - 提需求 (蓝色图标 + `#eff6ff` 背景)
   - 看进度 (绿色图标 + `#f0fdf4` 背景)
   - 咨询 (橙色图标 + `#fffbeb` 背景)
4. **段落标题** "热门推荐": 18px bold + "查看全部 >" 链接
5. **推荐卡片列表**: 左图右文横排卡片，每张包含:
   - 缩略图 (80x80px, 圆角 8px)
   - 标题 (15px medium) + 副标题 (12px secondary)
   - 价格 (16px bold primary) + 评分(金色星标 + 分数) + 分类标签
6. **底部 TabBar**: 固定定位，4 Tab

**关键组件**: Banner 轮播、快捷入口卡片、推荐列表卡片

**交互说明**:
- Tab 切换: hideEdge 动画
- 卡片点击 -> 进入 SKU 详情页
- "查看全部" -> 进入方案列表
- 快捷入口"选方案" -> 进入方案列表
- 快捷入口"提需求" -> 进入提交需求页

### 4.3 方案列表 -- `pages/sku/list`

**页面用途**: 浏览全部演出方案，支持搜索、分类筛选和排序。

**布局结构** (从上到下):
1. **导航栏** (sticky): 标题"方案列表" (17px bold)
2. **搜索栏**: 圆角胶囊 (9999px)，背景 `#f7f7f8`，高度 36px，搜索图标 + 占位文字
3. **分类 Tab 横向滚动**: 胶囊标签
   - 选中态: 背景 `#7c3aed`，白色文字
   - 未选中: 背景 `#f0f0f2`，文字 `#6b7280`
   - 分类: 全部、脱口秀、即兴喜剧、漫才、新喜剧、魔术喜剧、亲子喜剧
4. **筛选栏** (横向滚动): "价格排序"、"评分优先"、"咖位筛选"，带下拉箭头，边框胶囊
5. **方案卡片列表** (纵向): 每张卡片包含:
   - 顶部封面图 (全宽, 160px 高)
   - 标题 + 分类标签 (右对齐)
   - 副标题 (场景描述 + 时长)
   - 价格 + 评分 + 已售数量
6. **底部 TabBar**

**关键组件**: 搜索栏、分类胶囊标签、筛选标签、方案卡片(大图型)

**交互说明**:
- 搜索栏点击 -> 弹出搜索面板
- 分类标签切换 -> 筛选列表
- 卡片点击 -> 进入 SKU 详情页
- 下拉刷新、上拉加载

## 8. 适配规范

### 8.1 iPhone X+ 刘海屏安全区

- 顶部: 使用微信原生导航栏或自定义导航栏 + `env(safe-area-inset-top)` 适配
- 登录页和演员个人中心使用自定义导航栏(紫色渐变), 高度约 44px

### 8.2 底部安全区

- TabBar: `padding-bottom: env(safe-area-inset-bottom)`
- 固定操作栏: `padding-bottom: calc(12px + env(safe-area-inset-bottom))`
- 登录页底部: `padding-bottom: env(safe-area-inset-bottom, 20px)`

### 8.3 设计稿宽度

- 设计稿基准宽度: **375px** (iPhone 6/7/8)
- 所有 HTML 原型均使用 `max-width: 375px` 居中布局

### 8.4 rpx 与 px 换算

- 设计稿宽度 375px 对应微信小程序 **750rpx**
- 换算公式: **1px = 2rpx**
- 示例: 16px = 32rpx, 12px = 24rpx

| 设计稿 px | 小程序 rpx | 对应设计变量 |
|-----------|-----------|-------------|
| 10px | 20rpx | `--text-xs` |
| 12px | 24rpx | `--text-sm` |

```
                          ├── [Tab] 需求列表
                          │       └── "+" --> 提交需求
                          │       └── 卡片点击 --> 需求详情
                          │       │                   ├── "接受报价" --> 确认弹窗
                          │       │                   ├── "联系沟通" --> 对话
                          │       │                   └── "编辑" --> 编辑需求
                          │       └── "查看报价" --> 需求详情
                          └── [Tab] 个人中心(活动公司)
```

### 9.3 登录流程

```
用户打开小程序
  └--> 判断登录态
        ├── 未登录 --> 登录页 --> 微信一键登录
        │                         └--> 登录成功
```

```
                                                                                                           ├── 查看报价方案

                                                                                                                                   └--> 后续签约
```
