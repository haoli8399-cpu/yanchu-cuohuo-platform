# Handoff Protocol — 演立方/喜剧工厂

> Agent 任务完成后必须更新的交接协议。

---

## 何时更新

- ✅ 每次任务结束（无论完成或阻塞）
- ✅ Agent 被切换
- ✅ 遇到阻塞问题
- ✅ 里程碑完成

---

## Handoff 格式

```markdown
# HANDOFF — 项目状态

> 更新: YYYY-MM-DD HH:MM | Agent: [名称]

## 已完成
- [x] [Task] — [备注]

## 进行中
- [ ] [Task] — 进度X%

## 未完成
- [ ] [事项] — P0/P1/P2

## 阻塞问题
- ⚠️ [问题] — 原因/需要谁

## 修改文件
| 文件 | 操作 | 说明 |

## 下一步
1. [最紧急]
2. [次要]
3. [非紧急]
```

---

## 铁律

- **诚实**：阻塞就是阻塞，不要说"差不多"
- **完整**：列出所有修改文件
- **可操作**：下一步让接手Agent看了就能开始
- **同步写 Memory**：HANDOFF 更新同时写 OpenViking
