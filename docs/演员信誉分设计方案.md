# 演员信誉分设计方案

## 分值范围

**0-5分**，精确到小数点后2位。新入驻演员默认 **3.5分**（保护期）。

## 四大维度

| 维度 | 权重 | 含义 |
|------|:----:|------|
| **履约分** | 40% | 说到做到——接单后不取消、准时到场 |
| **质量分** | 35% | 活儿好不好——活动公司的多维度评价 |
| **活跃分** | 15% | 接单意愿——响应快、档期多 |
| **基础分** | 10% | 资料完整度——入驻时长、作品丰富度 |

## 计算公式

```
总分 = 履约分 × 40% + 质量分 × 35% + 活跃分 × 15% + 基础分 × 10%
```

每个维度取近6个月数据加权平均，**越近的交易权重越高**。

**时间衰减系数：**

| 距今时间 | 权重倍数 |
|---------|:--------:|
| 1个月内 | 1.0（完整权） |
| 1-3个月 | 0.7 |
| 3-6个月 | 0.4 |
| 超过6个月 | 不计入 |

---

## 加分项

| 行为 | 影响维度 | 分值 | 说明 |
|------|---------|:----:|------|
| 完成一场演出 | 履约分 | +1.0 | 正常完成演出 |
| 准时签到打卡 | 履约分 | +0.5 | 演出当天按时到场签到 |
| 活动公司评分≥4.5 | 质量分 | +0.5 | 单场评价 |
| 连续10场无违约 | 履约分 | +1.0 | 额外奖励 |
| 响应排期通知<30分钟 | 活跃分 | +0.2 | 快速确认 |
| 当月可接天数≥20天 | 活跃分 | +0.3 | 档期开放多 |
| 上传完整作品/案例 | 基础分 | +0.5 | 一次性奖励 |

## 扣分项

| 行为 | 影响维度 | 分值 | 说明 |
|------|---------|:----:|------|
| 确认排期后取消 | 履约分 | **-2.0** | 影响活动公司安排 |
| **演出当天临时取消** | 履约分 | **-5.0** | 严重违约，平台损失 |
| 迟到/未打卡 | 履约分 | -1.0 | 演出当天签到记录 |
| 活动公司评分<3.0 | 质量分 | -1.5 | 客户严重不满 |
| 响应排期通知>24小时 | 活跃分 | -0.3 | 长时间不回复 |
| 签约期内绕平台接单 | 履约分 | **-10.0** | 一票否决，信誉分归零 |

---

## 等级与权益

| 等级 | 分数范围 | 权益 | 说明 |
|:----:|:--------:|------|------|
| **S级** | ≥4.8 | ① 运营优先派单<br>② 活动公司搜索结果排首位<br>③ 可接高价SKU<br>④ "优选演员"标识 | 签约演员的目标等级 |
| **A级** | 4.5-4.79 | ① 运营优先派单<br>② 活动公司搜索结果靠前<br>③ 正常接单 | 签约演员标准等级 |
| **B级** | 4.0-4.49 | ① 正常派单 | 大多数演员在此区间 |
| **C级** | 3.5-3.99 | ① 正常派单<br>② 优先度降低 | 需改进 |
| **D级** | <3.5 | ① 限制接单（仅限低难度演出）<br>② 需平台审核后方可再接单 | 警告等级 |

---

## 新演员保护期

| 规则 | 说明 |
|:----|------|
| 初始分 | 新入驻演员默认 **3.5分** |
| 保护期 | 前3场演出**不扣分**（仅加分） |
| 保护期后 | 正常计入所有加减分项 |
| 保护期目的 | 避免新演员因前期磨合问题被差评一击致命 |

---

## 防刷机制

| 场景 | 处理方式 |
|:----|---------|
| 同活动公司多次合作 | 第3次起，该活动公司的评价权重降至**70%**，防止熟人刷分 |
| 连续差评且无合理解释的活动公司 | 运营人工审核，认定恶意差评后该评价不计入信誉分 |
| 演员拒绝接单后该需求立即消失 | 无法通过反复拒绝来"刷掉"不想接的单而不影响信誉分——**拒绝仍计入扣分** |
| 新账号刷分 | 需通过实名认证+作品审核才能入驻，注册门槛高 |

## 后台运营可配置

| 配置项 | 默认值 | 可调整范围 |
|-------|:------:|:---------:|
| 各维度权重 | 40/35/15/10 | 运营可调 |
| 扣分/加分值 | 见上表 | 运营可调 |
| 等级阈值 | S≥4.8 / A≥4.5 / B≥4.0 / C≥3.5 | 运营可调 |
| 新演员初始分 | 3.5 | 运营可调 |
| 保护期演出数 | 3场 | 运营可调 |
| 时间衰减系数 | 1.0/0.7/0.4 | 运营可调 |
