## 一、核心概念：双轨 + 双价

```
┌─────────────────────────────────────────────────┐
│                  演员评价体系                       │
│                                                   │
│  信誉分等级（S/A/B/C/D）     咖位等级（T0-T6）      │
│  衡量「靠不靠谱」             衡量「值不值钱」        │
│  系统自动计算                 运营手动设定+系统建议   │
│  影响：派单优先级、搜索排序    影响：演出费结算标准    │
│                                                   │
│  互补关系：咖位决定身价，信誉分决定能不能开高价       │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│                  价格双轨制                         │
│                                                   │
│  内部结算价（演员拿的）     外部报价（活动公司付的）   │
│  按咖位×时长×形式计算       按场景×人数打包          │
│  → 平台成本端               → 平台收入端            │
│                                                   │
│  毛利 = 外部报价 - 内部结算价                       │
└─────────────────────────────────────────────────┘
```

## 二、咖位等级（7级）

| 等级 | 名称 | 定义 | 对应信誉分门槛 | 认定方式 |
|:----:|------|------|:------------:|:--------:|
| **T0** | 顶流级 | 全国知名，票房保证，上过主流综艺且为主咖 | 不适用 | 人工认定 |
| **T1** | 综艺级 | 上过主流综艺（《脱口秀和Ta的朋友们》《喜剧之王单口季》等），有一定知名度但非顶流 | ≥4.5 | 人工认定+上传综艺截图 |
| **T2** | 头部演员 | 后仰签约核心，能独立撑专场，本地市场号召力强 | ≥4.5 | 系统建议+运营确认 |
| **T3** | 主力演员 | 稳定演出30+场，观众认可，能带新人 | ≥4.0 | 系统建议+运营确认 |
| **T4** | 成熟演员 | 独立完成演出，10+场经验 | ≥3.8 | 系统建议+运营确认 |
| **T5** | 成长演员 | 3-9场经验，培养观察期 | ≥3.5 | 系统建议+运营确认 |
| **T6** | 新人演员 | 入驻初期，0-2场，保护期内 | 默认3.5 | 运营手动选择 |

> T0/T1 不适用自动升降，纯人工认定。T2-T6 系统建议升降，运营确认生效。

## 三、内部结算价（平台→演员）

> 这是演员在平台上每完成一场演出后，平台结算给演员的金额。运营可在后台为单个演员自定义覆盖。

### 3.1 出演费（演员自带内容上台表演）

| 级别 | 基准价（20min） | 每+5min加价 | 计算规则 |
|:----:|:--------------:|:----------:|------|
| T0 | 面议 | 面议 | 每人每场单独协商 |
| T1 | ¥2,000 | ¥1,000 | 不够5分钟也按5分钟计 |
| T2 | ¥1,000 | ¥400 | 不够5分钟也按5分钟计 |
| T3 | ¥800 | ¥300 | 不够5分钟也按5分钟计 |
| T4 | ¥600 | ¥200 | 不够5分钟也按5分钟计 |
| T5 | ¥400 | ¥150 | 不够5分钟也按5分钟计 |
| T6 | ¥300 | ¥100 | 不够5分钟也按5分钟计 |

**时长取整规则**：实际演出不足20分钟按20分钟计；超出部分每5分钟为一档，不足5分钟按5分钟计。

**示例（T3演员）**：
- 演出5分钟 → ¥800（按20min基准）
- 演出22分钟 → ¥800 + ¥300 = ¥1,100
- 演出40分钟 → ¥800 + ¥300×4 = ¥2,000

### 3.2 定制段子创作费（编剧费）

| 级别 | 单价 | 规则 |
|:----:|:----:|------|
| T0 | 面议 | — |
| T1 | ¥1,000/min | 默认修改≤3次 |
| T2 | ¥800/min | 默认修改≤3次 |
| T3 | ¥600/min | 默认修改≤3次 |
| T4 | ¥400/min | 默认修改≤3次 |
| T5 | ¥300/min | 默认修改≤3次 |
| T6 | ¥200/min | 默认修改≤3次 |

> 创作费仅含编剧成本。超过3次修改的，每超1次加收创作费的10%作为改稿费（运营可配置）。

### 3.3 定制段子表演费（额外叠加）

| 级别 | 单价 | 说明 |
|:----:|:----:|------|
| T0 | 面议 | — |
| T1 | ¥3,000/min | 新段子需背稿+练习 |
| T2 | ¥200/min | 新段子需背稿+练习 |
| T3 | ¥150/min | 新段子需背稿+练习 |
| T4 | ¥100/min | 新段子需背稿+练习 |
| T5 | ¥80/min | 新段子需背稿+练习 |
| T6 | ¥50/min | 新段子需背稿+练习 |

> 表演费与出演费叠加计算。演员演自己写的定制段子 = 创作费 + 表演费 + 出演费。

### 3.4 综合示例（T2+T3拼盘，参照林芝方案）

```
演出：90min脱口秀拼盘
阵容：T2领衔(25min) + T3演员A(20min) + T3演员B(20min) + T3主持人(25min)
定制：每人3分钟林芝属地定制段子，演员自演

出演费：
  T2领衔 25min：1,000 + 400 = 1,400
  T3演员A 20min：800
  T3演员B 20min：800
  T3主持人 25min：800 + 300 = 1,100
  出演费小计：4,100元

定制创作费（3min × 3人）：
  T2: 800×3 = 2,400
  T3×2: 600×3×2 = 3,600
  创作费小计：6,000元

定制表演费（3min × 3人）：
  T2: 200×3 = 600
  T3×2: 150×3×2 = 900
  表演费小计：1,500元

内部结算总成本：11,600元
```

## 四、外部报价（活动公司→平台）

> 这是平台对活动公司的售卖价格。运营可在后台调整。

### 4.1 外出演出报价（核心业务线）

| 套餐 | 时长 | 人数 | 报价 | 适用场景 |
|:----:|:----:|:----:|-----:|------|
| 基础版 | 20min | 1人 | ¥3,000 | 小型活动/开业 |
| 标准版 | 40min | 2人 | ¥5,000 | 企业年会/中型活动 |
| 加长版 | 60min | 3人 | ¥7,000 | 大型活动/客户答谢 |
| 豪华版 | 90min | 5人 | ¥12,000 | 品牌发布/大型年会 |

> 以上为基础报价（对应T4/T5级别演员）。若客户要求T3及以上级别演员，单人加价按内部结算价差×系数1.5。例：客户要T2领衔→基础套餐+T2溢价。

**远程附加费**（在报价之上叠加）：
| 区域 | 附加费 |
|------|:------:|
| 成都市绕城内 | ¥0 |
| 绕城外、大成都内 | ¥500/人 |
| 成都市外 | ¥1,000/人/天 |

### 4.2 商演包场报价

| 场地 | 占用档期·周中 | 占用档期·周末 | 非占用档期 |
|------|:------------:|:------------:|:---------:|
| 王府井1号馆（310座） | ¥20,000 | ¥26,000 | ¥18,000 |
| 王府井2号馆（160座） | ¥16,000 | ¥20,000 | ¥15,000 |
| 招商大魔方（140座） | ¥15,000 | ¥15,000 | ¥15,000 |

> 以上为场地+基础演员打包价。指定更高咖位演员需补差价。

### 4.3 演出场次赞助报价

| 档位 | 价格 | 
|:----:|:----:|
| 周中场（周一至周五） | ¥2,000/场 |
| 周末场（周六至周日） | ¥4,000/场 |

含8项权益（品牌方可选配，详见AI模板文档）。

### 4.4 毛利结构参考

```
以「外出演出标准版 40min 2人」为例：

外部报价：¥5,000
内部结算（T4×2，各20min）：600×2 = ¥1,200
内部结算（T3×2，各20min）：800×2 = ¥1,600
内部结算（T2×1+T4×1）：1,000+600 = ¥1,600

平台毛利范围：¥3,400-3,800（68%-76%毛利率）

运营可自行调整报价和演员级别组合来控制毛利。
```

## 五、升降级规则

### 5.1 自动升级（系统建议 → 运营确认）

| 当前→目标 | 全部满足以下条件 |
|:--------:|------|
| T6→T5 | 完成≥3场 + 信誉分≥3.5 + 近3场无取消 |
| T5→T4 | 完成≥10场 + 信誉分≥3.8 + 近10场客户评分均分≥3.5 |
| T4→T3 | 完成≥30场 + 信誉分≥4.0 + 近20场客户评分均分≥4.0 |
| T3→T2 | 完成≥50场 + 信誉分≥4.5 + 近30场客户评分均分≥4.5 |
| T2→T1 | 完成≥80场 + 信誉分≥4.5 + 近30场客户评分均分≥4.5 + **需上过综艺**（人工认定） |

> T0/T1 不适用自动升降。T2→T1 额外要求综艺经历，系统仅作建议不自动触发。

### 5.2 自动降级（系统建议 → 运营确认）

| 触发条件 | 降级 |
|------|:--:|
| 信誉分连续3个月低于当前级别对应门槛 | 降1级 |
| 近10场客户评分均分 < 3.0 | 降1级 |
| 连续3次拒绝排期（无正当理由） | 降1级 |
| 演出当天临时取消 | 降2级 |
| 签约期内绕平台接单 | 直接降至T6 |

### 5.3 运营手动调级

运营可在后台随时调整任何演员的咖位（含T0/T1），操作需**填写调整原因+记录操作日志**。手动调级后3个月内系统不再自动建议该演员升降。

## 七、与信誉分体系的联动关系

```
信誉分等级（S/A/B/C/D） ≠ 咖位等级（T0-T6）

但是：
  ┌──────────────────────────────────────┐
  │  咖位升级的前置条件 = 信誉分达标       │
  │  咖位降级的触发条件之一 = 信誉分不达标  │
  │                                       │
  │  例：T4升T3 需要信誉分≥4.0            │
  │      如果 T3 演员信誉分跌破 4.0 持续   │
  │      3个月 → 系统建议降至T4            │
  └──────────────────────────────────────┘

两者并行运行，互不替代：
  - 信誉分决定「能不能接高价单」
  - 咖位决定「接什么价格的单」

一个演员可以 T3 咖位 + S级信誉分（身价中等、极其靠谱）
也可以 T2 咖位 + B级信誉分（身价高、但最近表现下滑）
```

## 八、对 AI 方案模板的影响

1. AI 生成方案时，根据需求难度匹配咖位区间，而非固定级别
2. 方案中标注「推荐咖位：T3及以上」，具体演员由运营排期时确定
3. 演员匹配权重新增「咖位匹配度」维度（权重0.10）
4. 内部结算价和外部报价在AI方案中均不显示具体金额（运营后台处理），AI只输出推荐级别范围
