```

  → Auth + RBAC
    → 项目脚手架（三端）
      → SKU CRUD API
        → SKU 后台页面 + SKU 浏览（Web + MP）
      → 需求提报 API
        → DeepSeek 集成
          → AI 方案查看（Web + MP）
          → 运营审核后台
            → 签约 API
            → 定金登记
              → 演员管理 + 咖位 + 信誉分
                → 排期分配
                  → 演员确认（MP）
                    → 订单状态机
                      → 签到打卡
                        → 尾款登记
                          → 结算统计
                            → 评价
                              → 运营工具（并行）
                              → 通知系统（并行）
                              → 价格配置（并行）
```
