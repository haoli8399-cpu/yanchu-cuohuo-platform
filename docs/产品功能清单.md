│                小程序 (uni-app + NutUI)            │
│              ├── 甲方/普通用户 (13项)              │
│              └── 个人艺人 (14项)                   │
│          活动公司基础功能与甲方共用                 │
└──────────────────┬───────────────────────────────┘
                   │
┌──────────────────▼───────────────────────────────┐
│          Web端 (Ant Design Pro + 自定义主题)        │
│   ├── 运营后台 (32项)                               │
│   └── 活动公司工作台 (14项)                          │
└──────────────────┬───────────────────────────────┘
                   │
┌──────────────────▼───────────────────────────────┐
│             后端 (Supabase自托管)                   │
