## 产品定位

**一句话描述：**
演立方 = AI 商演成交机器。输入需求 → 输出成交。

**产品类型：**
小程序 + PC Web + 运营后台

---

## 目标用户

| 用户角色 | 描述 | 核心痛点 |
|----------|------|---------|
| 活动公司 | 承接甲方演出需求，需要快速找到合适的演员方案 | 找演员靠微信问熟人，报价靠经验估，跟进靠脑子记 |
| 直接甲方 | 企业/品牌自己有演出需求，直接找方案 | 不懂行，不知道什么方案合理、多少钱合适 |
| 自营运营 | 平台内部运营人员，管理商机全流程 | 需求多，优先级乱，漏单丢单追不回 |
| 经纪公司 | 管理多名艺人的演出经纪公司 | 艺人分散管理难，派单靠电话，收入算不清 |
| 独立艺人 | 个人艺人，自己接单自己管理 | 一个人管排期、定价、客户，忙不过来 |
| 经纪公司艺人 | 签约在经纪公司旗下的艺人 | 不知道排期，结算流程不透明 |
| 平台管理员 | 平台运营方 | 看全局数据、管供应商、仲裁争议 |

---

## 核心价值主张

1. **一句话需求，快速成交一场演出** — 省去找演员的时间，10 分钟内看到方案和报价
2. **AI 经纪人「小演」** — 不是 chatbot，是主动推动成交的 AI 销售主管
3. **三层透明价格** — 甲方标准价 / 渠道价 (×0.7) / 成本价，不同角色看到该看的价格

## 核心功能范围

### ✅ 我们做

- AI 需求解析 → 方案匹配 → 报价生成 → 跟进提醒 → 成交
- 商机状态机（new → qualified → quoted → negotiating → won/lost）
- 标准化 SKU 方案库（脱口秀 / 即兴 / 魔术 / 漫才 / 亲子）
- 三层价格体系（甲方/渠道/成本）
- 经纪人销售作战台（商机队列 + AI 方案 + 跟进）
- 艺人排期 / 签到 / 结算 / 信誉

### ❌ 我们不做

- 票务系统 — 非核心，已有成熟方案
- 场地租赁 — 非核心能力
- 活动策划全案 — 非标，难以标准化
- 社交/社区 — 不需要
- 比价平台 — 与京东自营定位冲突
- 免费模式 — 破坏演员生态
- 演唱会/明星经纪 — 只聚焦脱口秀/喜剧/小剧场商演

---

## 产品原则

1. **成交优先** — 所有功能先问：能不能帮用户更快成交？
2. **AI 驱动，不是 AI 贴皮** — 小演是产品的灵魂，不是外挂窗口
3. **自营先行，平台后开** — 先自己跑通闭环，再开放第三方
4. **不做大而全，做深而准** — 砍功能比加功能更需要勇气

## 版本规划

### Phase 1：AI 辅助商演成交 — 当前
- 自营运营闭环（需求 → 商机 → 报价 → 跟进 → 成交）
- 活动公司/甲方浏览方案、提交需求
- 小程序 5 Tab + PC 供应商工作台 + Admin 运营后台
- 小演 AI 悬浮助手（洞察 + 话术）

### Phase 2：需求入口升级
- 自然语言需求 → AI 自动匹配
- 小程序 AI 对话升级

### Phase 3：平台开放 + Agent 商业化
- 经纪公司/独立艺人入驻
- AI Agent 订阅付费（¥199/499 月）
- 平台抽佣
- 争议仲裁
