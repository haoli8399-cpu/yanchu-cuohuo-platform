### 第一优先（P0 — 阻塞发布）

5. **在 `sku/list/index.vue` 卡片中使用供应商标识** — 已有 CSS 类，只需模板接入
6. **实现 8 宫格功能网格** — 替换现有列表菜单

### 第三优先（P2 — 体验优化）
